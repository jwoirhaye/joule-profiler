# Usage

The profiler can be used either via the CLI or through the library, which exposes traits and data structures for programmatic access.  

[Using the CLI](cli.md) provides a quick, turnkey solution that is easy to get started with and enable minimal configuration options.

On the other hand, [using the library](lib.md) offers more flexibility and configuration options for users who need fine-grained control, while remaining straightforward and user-friendly to integrate easily into your Rust applications.
